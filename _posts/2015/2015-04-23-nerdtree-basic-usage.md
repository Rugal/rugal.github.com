Here I list same commands I think is most useful.  


* `?`     help
* `C`     move into this folder
* `u`     move to parent folder
* `cd`    similar as `cd` in shell, change `PWD` into this folder
* `m`     open NERDTree menu
* `I`     Toggle hidden files
* `r`     Refresh file list
